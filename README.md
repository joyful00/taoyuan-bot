## ✨ 核心功能 (Features)

### 👗 1. 突破限制：无限虚拟衣柜 (全静默秒切)

- **场景自动识别**：进入农场自动换“农民套”，进入矿洞瞬间切“战斗套”，去商店秒换“商人套”。
- **无感极速换装**：跨过繁琐的换装动画，切地图的瞬间装备就已经穿好，**全程无弹窗、不卡顿**。
- **纯中文配置**：自带强大的物品词典，只需在控制面板输入装备的中文名即可自由搭配，轻松突破官方只能存 3 套衣服的限制！

### 👁️ 2. 降维打击：矿洞上帝之眼 (全图透视)

- **开局即全图**：进入矿洞后，无需点击任何方块，直接全图透视所有隐藏物品。
- **安全扫雷**：👿 怪物标红、🚪 出口标绿、💎 宝藏/矿石标黄。清清楚楚，绝不翻车，拒绝无效消耗体力！

### 🎣 3. 极速引擎：全自动满级钓鱼

- **精准自瞄**：彻底解放双手，自动抛竿、自动收线。
- **动态追踪**：完美模拟高频按键，死死咬住变色目标区域，百发百中，绝不脱钩！

### 👨‍🌾 4. 全场景自动化挂机

- **农田与村庄**：一键处理繁琐的连续交互操作。
- **商店大采购**：自带收益计算，自动扫货与抛售，做无情的赛博资本家。

## ⚙️ 可视化控制台 (Configuration)

告别繁琐的代码修改，一切皆可点！

1. **专属悬浮面板**：游戏内注入专属的蓝色小机器人悬浮窗，支持全屏自由拖拽，不遮挡游戏画面。
2. **配置永久保存**：在面板里设置好的所有参数（套装方案、开关状态、滑块参数）都会自动保存在你的浏览器中，刷新网页不丢失，关机重启依然生效！
3. **实时动态调参**：钓鱼力度感觉不对？矿洞今天不想开透视了？直接在面板里拖动滑块或点击开关，重新进入对应场景后立即生效，随心所欲。

---

## 🚀 安装指南 (Installation)

1. 安装浏览器脚本管理器插件：推荐使用 [ScriptCat (脚本猫)](https://docs.scriptcat.org/) 或 [Tampermonkey (油猴)](https://www.tampermonkey.net/)。
2. 点击本页面（或 GreasyFork）的 **“安装脚本”** 按钮。
3. 刷新《桃源乡》游戏页面，即可在游戏右侧看到专属的助手悬浮球，点击展开即可开启你的修仙之旅！
